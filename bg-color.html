Add Background Color

To add a background color to the entire HTML document, you can use CSS either inline, internally, or externally.

  • Inline Styling (within the HTML tag):

    • <html lang="en" style="background-color: lightblue;">

  • Internal Styling (within the <style> tag in the <head> section):

    • <head>
    <style>
    body {
    background-color: lightblue;
    }
    </style>
    </head>

  • External Styling (using an external CSS file):

    • /* styles.css */
    body { background-color: lightblue; } Choose
